# MoldBios
A basic bios written in x86 assembly and C that runs in protected mode for QEMU.

## Roadmap

### [-] VGA Driver (Working on)
### [ ] PS/2

### [ ] Floppy
### [ ] Booting from floppy, hard drive or cd-rom
